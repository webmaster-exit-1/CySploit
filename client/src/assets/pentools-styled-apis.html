API Services

The following services offer free or paid access to valuable security data.

API Service Description Type Notes
Shodan Search engine for Internet-connected devices. Uncovers servers, webcams, routers, and more. Tiered Free API has limits. Academic plans available. Excellent for reconnaissance.
Censys Search engine that enables researchers to discover security vulnerabilities across devices. Tiered Free academic access. Commercial plans available.
HaveIBeenPwned Check if email accounts or passwords have been compromised in data breaches. Free/Paid Free for individuals. API access requires subscription.
VirusTotal Analyze suspicious files and URLs for malicious content. Tiered Free tier with limits. Enterprise plans available.
WhatIsMyBrowser User agent parsing, browser and device detection. Tiered Free tier with very limited requests per day.
GreyNoise Analyses Internet-wide scan and attack traffic. Tiered Community access available. API key needed for full features.
URLScan.io Scans and analyzes websites by simulating regular user visits. Free Rate limits apply, but generous free tier.
Pulsedive Threat intelligence made easy with historical context. Tiered Free community edition. Pro features require subscription.
